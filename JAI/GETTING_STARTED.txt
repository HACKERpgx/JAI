QUICK START (3 STEPS)
================================================================================

1. INSTALL DEPENDENCIES
   
   Open PowerShell in this folder and run:
   
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt

2. CONFIGURE API KEYS
   
   Your .env file should already have:
   - OPENAI_API_KEY
   - NEWS_API_KEY
   - OPENWEATHER_API_KEY
   
   Optional settings:
   - SPEAK_RESPONSES=true  (enable spoken responses)
   - TTS_VOICE=David       (voice name)

3. START JAI
   
   Option A - Server only:
   python jai_assistant.py
   
   Option B - Voice interface (RECOMMENDED):
   # Terminal 1:
   python jai_assistant.py
   
   # Terminal 2:
   python voice_client.py
   
   Then say: "Hey JAI, what's the weather?"

AUTO-START ON WINDOWS
================================================================================

To make JAI start automatically when you log in:

1. Open PowerShell as Administrator
2. Navigate to this folder
3. Run: .\setup_windows_startup.ps1

To remove auto-start:
.\setup_windows_startup.ps1 -Remove

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Module not found" errors
Solution: pip install -r requirements.txt

Problem: TTS not working
Solution: Set SPEAK_RESPONSES=true in .env
         Try different voice: TTS_VOICE=Zira

Problem: Microphone not detected
Solution: Check Windows microphone permissions
         Test with: python test_speech.py

Problem: "Cannot connect to JAI server"
Solution: Make sure server is running: python jai_assistant.py

Problem: PyAudio installation fails
Solution: pip install pipwin
         pipwin install pyaudio
