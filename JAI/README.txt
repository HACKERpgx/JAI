JAI Voice Assistant
==================

A voice-enabled AI assistant with support for multiple languages and voice commands.

## Installation

1. Run the JAI_Voice_Installer.exe
2. Follow the installation wizard
3. The installer will automatically install all required Python packages

## Requirements

- Windows 7 or later
- Python 3.8 or later
- Internet connection (for voice recognition and AI features)

## Voice Commands

- "Hey JAI, what's the weather?" - Get weather information
- "Hey JAI, set a reminder" - Set a reminder
- "Hey JAI, play some music" - Play music
- "Hey JAI, open calendar" - View calendar

## Default Voice

The default voice is set to David (English - US). You can change this in the application settings.

## Troubleshooting

If you encounter any issues:
1. Make sure your speakers are not muted
2. Check that the correct audio output device is selected in Windows
3. Ensure you have the latest Windows updates installed
